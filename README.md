# JPEG-AI-framework
JPEG AI is a next-generation image compression framework being developed by the Joint Photographic Experts Group to use artificial intelligence and deep learning for image coding. It is intended as a future evolution beyond:  JPEG, JPEG2000 and JPEG XL.  with much higher compression efficiency and perceptual quality.
